氣泡排序法（Bubble Sort）

氣泡排序是一種簡單的比較式排序演算法，它反覆地在待排序序列中從頭到尾進行多次對比和交換。每次遍歷都會把較大的元素 "冒泡" 到序列的末端，因此最終會將資料按照 由小到大 的順序排列。

演算法步驟

  1. 從第一個元素開始，逐一比較相鄰的兩個元素。
  2. 若前者比後者大，則交換兩者位置。
  3. 繼續對後面的元素進行同樣動作，完成一次遍歷最多可將一個最大元素移到最右端。
  4. 重複以上過程，忽略已排序好的末端部分，直到整個序列有序。

數字排序動畫示範

下方展示了一組隨機整數如何利用氣泡排序逐步由小到大排列。可使用控制按鈕暫停、步進或重新生成資料，並切換語言閱讀說明。

程式碼

// 雙迴圈氣泡排序示範，將數字由小到大排序
// 使用範例：
// let arr = [34, 7, 23, 32, 5, 62, 14];
// bubbleSort(arr, visualize);
function bubbleSort(arr, visualize) {
    let n = arr.length;
    let swapped;
    do {
        swapped = false;
        for (let i = 0; i < n - 1; i++) {
            visualize(i, i + 1);
            if (arr[i] > arr[i + 1]) {
                // 交換位置
                [arr[i], arr[i + 1]] = [arr[i + 1], arr[i]];
                swapped = true;
                visualize(i, i + 1, true);
            }
        }
        n--; // 最右邊數值已經就定位
    } while (swapped);
}

應用簡述

氣泡排序屬於穩定排序演算法，實作簡單但在大型數據集上效率較低。以下是常見應用的簡要說明：

  • 教學用途：用於說明基本排序概念和演算法步驟。
  • 小型資料集：當資料量很小且開發時間為主要考量時。
  • 嵌入式/初級系統：在記憶體受限且程式碼簡潔時。
